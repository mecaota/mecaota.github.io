PC making and power user

親いわくパソコンに触り始めたのは3歳頃だそうです。自分でPCを立ち上げて某パンの顔をしたヒーローのゲームをしていたそうです。そこにルーツがあるのかは知りませんが、ずっとPCを触ることが好きで、遅いPCを動かすためにレジストリを触ったり、パーツ換装したり、今でもPC自作を嗜んでいます。

Cyclist

自転車乗りです。住んでる周りを100km程度の範囲で走ってみたり、ファンライドイベントで各地を走ってみたり。ママチャリに乗っていたときから自転車が好きでした。その愛は大学1年次に創設した大学自転車部に現れており、現在も後輩たちへその愛は引き継がれています。

Traveler

旅行するのが趣味です。小さい頃から月に1回はどこか高速道路に乗るくらいの長距離の遠出をしないと泣いてたくらい好きです。現在も夏休みや春休みの長期休みを使って南は関西、北は道東エリアなどレンタカーを借りたり、18切符を使ったりして旅をしています。

Creditcard Collecter

小学生の時から走りの非接触電子マネーカードを所持していたり、大手コンビニのポイントカードを集めていたりしましたが、現在もその流れでクレジットカード集めが趣味です。大学3年修了次で既にカード枚数は10枚に達し、現在も増え続けています。使っているかは謎です。

Gadget Collecter

スマホやタブレットを買い漁ったり、変な端末や周辺機器があれば思わず手に取ってしまう癖があるようです。現在も机の周りは増殖の一途を辿り、よくわからないものが増えていきます。
